AI-Generated Code Detection API

This powerful API analyzes GitHub repositories and websites to determine whether a project was likely generated or heavily assisted by AI coding tools. By providing a repository URL or website link, the system scans the source code and evaluates multiple structural and behavioral signals.

Using advanced techniques such as AST parsing, JavaScript bundle analysis, entropy analysis, code repetition metrics, and LLM fingerprint detection, the API identifies patterns commonly found in AI-generated codebases.

After processing, the system returns a vibe score, detected frameworks, and detailed signals showing why the project may be AI-generated. This helps developers, researchers, and organizations audit repositories, understand code origins, and evaluate AI influence in modern software projects.
